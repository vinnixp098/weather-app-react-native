# App de previsão do tempo

Este é o meu primeiro app desenvolvido em React Native, criado para fornecer previsões de clima através de uma API de clima. Para lidar com as requisições à API, utilizei a biblioteca Axios, pois ela facilita a comunicação com o servidor. 

## Tecnologias Utilizadas

- **React Native**: Biblioteca JavaScript para criação de aplicativos multiplataforma.
- **Axios**: Biblioteca para realizar requisições HTTP.
- **CSS Puro**: Utilizado para estilizar o projeto de forma personalizada.
- **Expo**: Ferramenta que simula o ambiente de desenvolvimento mobile.

## Funcionalidades

- Exibe previsões de clima baseadas no nome da cidade fornecida pelo usuário.
- Interface de usuário estilizada com CSS puro para um visual limpo e moderno.

## Sobre o Projeto

Este projeto representa o início de minha jornada no desenvolvimento mobile com React Native. É o primeiro de muitos projetos que planejo desenvolver utilizando este framework poderoso. A ideia é explorar e aprender cada vez mais sobre React Native e suas possibilidades, construindo aplicações mobile que visam resolver problemas do dia a dia.

## Como Executar o Projeto

1. Clone o repositório:
   ```bash
    git clone https://github.com/vinnixp098/weather-app-react-native.git
   
2. Navegue até o diretório:
   ```bash
     cd weather-app-react-native
   
3. Instale as dependências:
   ```bash
     npm install
   
4. Inicie o servidor de desenvolvimento:
   ```bash
     npx expo start

5. Abra o Expo e aponete a câmenra do celular para o QR Code que aparece na tela.
